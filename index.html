VSC-tools is a set of Python libraries and scripts that are commonly used within our organisation.

REQUIREMENTS

VSC-tools requires Python 2.4 (or a more recent 2.x version) to be available. For the LDAP tools, obviously, you need python-ldap.

COMPONENTS

VSC-tools consists of multiple separate packages.

vsc-base

  • vsc.fancylogger: an enhanced logging tool
  • vsc.utils.affinity: setting cpu affinity
  • vsc.util.asyncprocess: allow asynchronous subprocesses to be executed
  • vsc.utils.cache: file-based caching of timestamped data
  • vsc.utils.daemon: a generic daemon class (public domain software reused from Sander Marechal
  • vsc.utils.dateandtime: convenience functions for dealing with times and dates
  • vsc.utils.generaloption: an enhanced option parser, offering three levels: configuration file, environment variables and command line options
  • vsc.utils.IPy: handling of IPV4 and IPV6 addresses and networks
  • vsc.utils.mail: convenience wrapper around standard python mail functionality
  • vsc.utils.missing: various useful functions and classes (e.g., monoids, monois based dictionaries, ...)
  • vsc.utils.nagios: tools to provide functionality for exiting with a correct status that can be read by icinga/nagios
  • vsc.utils.patterns: common software design patterns
  • vsc.utils.run: execution of command
